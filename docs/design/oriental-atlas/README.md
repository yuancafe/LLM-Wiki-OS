# Oriental Atlas Design Reference

This directory contains the approved design baseline for the graph HTML refactor.

- `oriental-editorial-atlas.html`: approved prototype to match visually and structurally
- `design-brief.md`: product and interaction brief
- `DESIGN.md`: visual system and component guidance
- `reviews/result-review.md`: prototype review notes and responsive findings

Implementation work should treat this directory as the local source of truth. Design workspace copies are not implementation branch contracts unless they are synchronized here.
